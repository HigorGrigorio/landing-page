databricks

Databricks is an Apache Spark-based analytics platform. Designed with the founders of Apache Spark, with Databricks we have streamlined workflows and an interactive workspace that enables collaboration between data scientists, data engineers and business analysts.

With support for Python, Scala, R and SQL, as well as libraries and deep learning structures like TensorFlow, Pytorch and Scikit-learn, using Databricks we eliminate all the hardness and complexity to get a Spark cluster.Your notebooks provide an uninterrupted, seamless management experience thanks to integration with leading cloud providers, including Amazon AWS and Microsoft Azure.
